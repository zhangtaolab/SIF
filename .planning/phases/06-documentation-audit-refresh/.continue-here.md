## Critical Anti-Patterns

| Pattern | Description | Severity | Prevention Mechanism |
|---------|-------------|----------|---------------------|
| Untracked file merge conflict | Worktrees create files that may conflict with untracked files on main (e.g., scripts/generate_cli_ref.py, SUMMARY.md files). Git merge aborts with "untracked working tree files would be overwritten." | blocking | Before merging any worktree, run `git status` and remove/move untracked files that would be overwritten. Check `git diff --name-only --diff-filter=U` after merge attempts. |
| Worktree branch cleanup failure | `git worktree remove` may fail if the worktree is "locked" (agent still running). Branches may persist after removal attempts. | advisory | Use `git worktree remove --force` and `git branch -D` aggressively. Check `git worktree list` after cleanup. |
| Stale _auto_chain_active flag | Previous --auto runs may leave `workflow._auto_chain_active: true` in config, causing unwanted auto-advance on next phase. | advisory | Execute-phase workflow clears this flag at init when no --auto flag is present. |
