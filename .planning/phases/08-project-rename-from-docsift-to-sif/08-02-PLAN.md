    The migration logic (per D-02):
    1. Check if old model cache path exists: `~/.local/share/docsift/models/`
    2. Check if new model cache path does NOT exist: `~/.local/share/sif/models/`
    3. If both conditions true, rename old to new using `old_path.rename(new_path)`
    4. Print message: `Migrated: ~/.local/share/docsift -> ~/.local/share/sif`

    Insert this code at the beginning of `main()`, before `cli()`:

    ```python
    def main() -> None:
        """Main entry point."""
        # Migrate old docsift model cache if present
        old_model_dir = Path.home() / ".local" / "share" / "docsift" / "models"
        new_model_dir = Path.home() / ".local" / "share" / "sif" / "models"
        if old_model_dir.exists() and not new_model_dir.exists():
            new_model_dir.parent.mkdir(parents=True, exist_ok=True)
            old_model_dir.rename(new_model_dir)
            console.print("[green]Migrated: ~/.local/share/docsift -> ~/.local/share/sif[/green]")
        
        cli()
    ```
