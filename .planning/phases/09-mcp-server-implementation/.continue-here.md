# BLOCKING CONSTRAINTS — Read Before Anything Else

- [x] **CONSTRAINT: MCP 2025-11-25 Streamable HTTP replaces HTTP+SSE** — The new standard uses a single `/mcp` endpoint with POST/GET, not separate `/mcp/v1/sse` + `/mcp/v1/messages`. Old endpoints must NOT be implemented.
- [x] **CONSTRAINT: SQLite DDL triggers implicit commit** — Creating tables inside a transaction that later rolls back will NOT roll back the DDL. Never assert DDL rollback in SQLite tests.
- [x] **CONSTRAINT: stdout must never contain non-JSON output in stdio mode** — All logs must go to stderr only. Any non-JSON stdout will break MCP clients.

## Critical Anti-Patterns

| Pattern | Description | Severity | Prevention Mechanism |
|---------|-------------|----------|---------------------|
| Long-lived DB connections | Holding SQLite connection across tool calls causes concurrency issues | advisory | New DatabaseConnection per tool call (D-04) |
| Wildcard CORS defaults | `cors_origins=["*"]` is insecure per MCP spec | advisory | Default to `["http://localhost:3000", "http://127.0.0.1:3000"]` (D-14) |
| Blocking stdout in stdio mode | print() or logging to stdout corrupts JSON-RPC stream | blocking | All logging goes to stderr; stdout is JSON-RPC only |

<decisions_made>

- **D-01:** Reuse legacy `mcp/protocol.py` Pydantic models — models are well-defined and SIF-specific
- **D-02:** Use OOP `ToolHandler` ABC pattern — each tool is a class with name/desc/schema/handle
- **D-03:** Directory structure A (functional layering) — handlers.py + backend.py + transports/
- **D-04:** New `DatabaseConnection` per tool call — safety first, no concurrency issues
- **D-05:** `SearchBackend` methods are `async` — matches async transport semantics
- **D-06:** Cache embedder via `EmbeddingManager` singleton — balances safety with performance
- **D-07:** Full MCP lifecycle (initialize → initialized → tools/list → tools/call)
- **D-08:** Tiered error handling — fatal→exit, recoverable→JSON-RPC error response
- **D-09:** Full notification support — initialized, progress, cancelled
- **D-10:** Streamable HTTP (2025-11-25) — single `/mcp` endpoint, POST/GET, session-aware
- **D-11:** No backwards compatibility with old HTTP+SSE transport
- **D-12:** Default bind to `127.0.0.1`
- **D-13:** Validate `Origin` header, reject invalid with 403
- **D-14:** CORS defaults non-wildcard
- **D-15:** Resource URI: `docs://{doc_id}` with optional `?from_line=N&max_lines=M`
