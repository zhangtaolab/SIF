       - `async def hybrid_search(self, query: str, collections: list[str] | None = None, limit: int = 10, min_score: float = 0.0) -> list[protocol.SearchResult]`:
         - Inside to_thread: create `DatabaseConnection`, open transaction, build `SearchOptions(limit=limit, min_score=min_score, include_content=False, include_highlights=True)`
         - If collections provided: resolve names to IDs via `CollectionRepository.get_by_name()`, set `options.collection_ids`
         - Get embedder: call `self._embedding_manager.load_model()` (lazy, cached), then `self._embedding_manager.embed(query)` to get query embedding

         - Call `searcher.search(query, options)` to get `list[CoreSearchResult]`
         - Map each `CoreSearchResult` to `protocol.SearchResult` (fields: doc_id, path, title, content, score, highlights, metadata)
         - Return list of protocol.SearchResult
       - `async def get_document(self, path_or_docid: str, from_line: int | None = None, max_lines: int | None = None) -> protocol.Document | None`:
         - Inside to_thread: open transaction, try `DocumentRepository.get_by_id(path_or_docid)`, if not found try `get_by_path()` across all collections
         - If found and line slicing requested: split content by "\n", slice from from_line for max_lines, set line_start/line_end
         - Map to `protocol.Document` and return
       - `async def get_documents_by_pattern(self, pattern: str, max_bytes: int | None = None) -> tuple[list[protocol.Document], list[str]]`:
         - Inside to_thread: open transaction, list all collections, for each collection list documents via `DocumentRepository.list_by_collection()`
         - Use `fnmatch.fnmatch(doc.path, pattern)` or `fnmatch.fnmatch(doc.doc_id, pattern)` to filter
         - Track total bytes; if max_bytes and adding next doc would exceed, add error string "Max bytes limit reached, skipping {doc.path}" and skip
         - Map each matching doc to `protocol.Document`, return (documents, errors)
       - `async def get_status(self) -> tuple[list[protocol.CollectionInfo], int]`:
         - Inside to_thread: open transaction, get all collections via `CollectionRepository.list_all()`
         - For each collection, count documents (use `DocumentRepository` or raw SQL `SELECT COUNT(*) FROM documents WHERE collection_id = ?`)
         - Build `protocol.CollectionInfo(name=coll.name, document_count=count, last_updated=coll.last_indexed_at.isoformat() if coll.last_indexed_at else None)`
         - Get total document count via `SELECT COUNT(*) FROM documents`
         - Return (collection_infos, total_count)
