## Objective

Implement four MCP tool handlers (query, get, multi_get, status) as concrete ToolHandler classes. Each handler validates input via Pydantic, calls SearchBackend methods, and returns properly formatted ToolsCallResult. Wire tool registration into MCPServer. Implement resources/read inline in server.py (unified dispatch pattern — no separate ResourceHandler ABC).

Purpose: Expose real search and retrieval capabilities through the MCP protocol.
Output: 4 working tool handlers registered with MCPServer + resources/read dispatch.

From `src/sif/mcp/protocol.py`:
```python
class ToolsCallResult(BaseModel):
    content: list[ToolContentItem] = Field(default_factory=list)
    isError: bool = False

class ToolContentItem(BaseModel):
    type: Literal["text", "image", "resource"] = "text"
    text: Optional[str] = None
    data: Optional[str] = None
    mimeType: Optional[str] = None

class MCPTool(BaseModel):
    name: str
    description: str
    inputSchema: ToolInputSchema = Field(default_factory=ToolInputSchema)

class QueryInput(BaseModel):
    query: str
    collections: Optional[list[str]] = None
    limit: int = 10
    min_score: float = 0.0

class GetInput(BaseModel):
    path_or_docid: str
    from_line: Optional[int] = None
    max_lines: Optional[int] = None

class MultiGetInput(BaseModel):
    pattern: str
    max_bytes: Optional[int] = None

class StatusOutput(BaseModel):
    collections: list[CollectionInfo] = Field(default_factory=list)
    total_documents: int = 0
```

From `src/sif/mcp/backend.py` (from 09-02):
```python
class SearchBackend:
    async def hybrid_search(self, query: str, collections: list[str] | None = None, limit: int = 10, min_score: float = 0.0) -> list[SearchResult]
    async def get_document(self, path_or_docid: str, from_line: int | None = None, max_lines: int | None = None) -> Document | None
    async def get_documents_by_pattern(self, pattern: str, max_bytes: int | None = None) -> tuple[list[Document], list[str]]
    async def get_status(self) -> tuple[list[CollectionInfo], int]
```
