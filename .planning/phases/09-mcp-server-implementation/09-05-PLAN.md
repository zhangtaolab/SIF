Purpose: Enable remote MCP access via HTTP with secure defaults.
Output: Working FastAPI app with `/mcp` endpoint and `run_http_server` entry point.

    1. Create `src/sif/mcp/transports/http.py` with:

    **Constants:**
    - `DEFAULT_CORS_ORIGINS = ["http://localhost:3000", "http://127.0.0.1:3000"]`
    - `DEFAULT_HOST = "127.0.0.1"`
    - `DEFAULT_PORT = 3000`

    **Pydantic models:**
    - `class MessageRequest(BaseModel)`: jsonrpc="2.0", id: str|int|None, method: str|None, params: dict|None

    **Session management:**
    - Maintain an in-memory dict `_sessions: dict[str, Any]` mapping session IDs to session state
    - `def _generate_session_id() -> str`: return `uuid.uuid4().hex`
    - On POST /mcp: if request has `MCP-Session-Id` header, validate it exists in _sessions; if not present or invalid, generate new session ID
    - On GET /mcp for SSE: if request has `MCP-Session-Id` header, use existing session; else generate new session ID
    - Return `MCP-Session-Id` header in ALL POST and GET responses

    **create_app function:**
    - `def create_app(server: MCPServer, cors_origins: list[str] | None = None) -> FastAPI:`
    - `origins = cors_origins if cors_origins is not None else DEFAULT_CORS_ORIGINS`
    - Create FastAPI app
    - Add CORSMiddleware with allow_origins=origins, allow_credentials=True, allow_methods=["*"], allow_headers=["*"]
    - Add custom middleware for Origin validation (after CORS): if Origin header present and not in origins, return 403
    - POST /mcp endpoint:
      - Accept `MessageRequest` body
      - Read `MCP-Session-Id` header from request; if missing or unknown, generate new session ID and store in _sessions
      - If request.id is None (notification): parse as JsonRpcNotification, call `await server.handle_notification(notification)`, return JSONResponse(status_code=202, content={}, headers={"MCP-Session-Id": session_id})
      - Else: parse as JsonRpcRequest, call `await server.handle_request(rpc_request)`, return JSONResponse(content=response.model_dump(exclude_none=True), headers={"MCP-Session-Id": session_id})
      - Handle validation errors: return 400 with INVALID_REQUEST error, include MCP-Session-Id header
    - GET /mcp endpoint:
      - Read `MCP-Session-Id` header; if missing, generate new session ID
      - Return StreamingResponse with SSE stream, headers include `MCP-Session-Id`
      - Event generator yields keepalive pings every 30 seconds: `f"data: {json.dumps({'ping': True})}\n\n"`
      - Stops when client disconnects (use request.is_disconnected())
      - Media type: "text/event-stream"
    - Optional /health endpoint: return {"status": "ok"}

    **HTTPTransport class:**
    - `__init__(self, server: MCPServer, host: str = DEFAULT_HOST, port: int = DEFAULT_PORT, cors_origins: list[str] | None = None)`
    - `def run(self) -> None`: create app via create_app, call `uvicorn.run(app, host=self.host, port=self.port, log_level="info")`

    **run_http_server function:**
    - `def run_http_server(db_path: str, host: str = DEFAULT_HOST, port: int = DEFAULT_PORT, cors_origins: list[str] | None = None) -> None:`
    - Create SearchBackend, MCPServer, register default tools
    - Create HTTPTransport and call transport.run()

    2. Update `src/sif/mcp/transports/__init__.py` to export `HTTPTransport`, `create_app`, `run_http_server`.
    3. Update `src/sif/mcp/cli.py`:
       - `run_http_server(db_path: str, host: str, port: int, cors_origins: list[str] | None)` should import from `.transports.http` and call the transport's run_http_server
    4. Ensure all imports use correct paths. Use `from __future__ import annotations`.

| T-09-10 | Spoofing | Origin header spoofing | mitigate | Custom middleware validates Origin against allowlist; rejects with 403 |
| T-09-11 | Information Disclosure | Server binds to 0.0.0.0 exposing to network | mitigate | DEFAULT_HOST is "127.0.0.1"; --host allows override with explicit opt-in |
| T-09-12 | Denial of Service | Oversized JSON-RPC request body | mitigate | FastAPI/Pydantic validates body size; Pydantic model validation rejects malformed |
