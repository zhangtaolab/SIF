    1. Create `src/sif/mcp/transports/stdio.py` with `StdioTransport` class:
       - `__init__(self, server: MCPServer, stdin: TextIO | None = None, stdout: TextIO | None = None)`:
         - Store server, stdin (default sys.stdin), stdout (default sys.stdout)
         - `_running = False`, `_read_lock = asyncio.Lock()`, `_write_lock = asyncio.Lock()`
       - `async def _read_message(self) -> dict[str, Any] | None`:
         - Acquire `_read_lock`
         - Use `asyncio.get_event_loop().run_in_executor(None, self.stdin.readline)` to read non-blocking
         - If empty string/EOF: return None
         - Strip line, if empty return None (skip blank lines)
         - Try `json.loads(line)`; on JSONDecodeError return `{"jsonrpc": "2.0", "id": None, "error": {"code": -32700, "message": f"Parse error: {e}"}}`
       - `async def _write_message(self, message: dict[str, Any]) -> None`:
         - Acquire `_write_lock`
         - `json.dumps(message, ensure_ascii=False)`
         - Use `run_in_executor` to write line + "\n" to stdout and flush
       - `async def _process_message(self, message: dict[str, Any]) -> dict[str, Any] | None`:
         - If message has no "id" field (notification): parse as `JsonRpcNotification`, call `await self.server.handle_notification(notification)`, return None
         - Else: parse as `JsonRpcRequest`, call `await self.server.handle_request(request)`, return response dict
         - Catch exceptions: log error, return error response with request id
       - `async def run(self) -> None`:
         - Set `_running = True`, log "Stdio transport started"
         - While `_running`:
           - Read message
           - If None: break (EOF)
           - Process message
           - If response is not None: write response
         - Catch asyncio.CancelledError: log "Transport cancelled", re-raise
         - Catch Exception: log exception
         - Finally: set `_running = False`, log "Stdio transport stopped"
    2. Add `run_stdio_server(db_path: str) -> None` async function:
       - Create `SearchBackend(db_path)`
       - Create `MCPServer(backend)`
       - Register default tools: `server.register_tools(create_default_tools(backend))`
       - Create `StdioTransport(server)`
       - `await transport.run()`
    3. Add `run_stdio_server_sync(db_path: str) -> None` synchronous wrapper:
       - `asyncio.run(run_stdio_server(db_path))`
    4. Update `src/sif/mcp/transports/__init__.py` to export `StdioTransport`, `run_stdio_server`, `run_stdio_server_sync`.
    5. Update `src/sif/mcp/cli.py`:
       - `run_stdio_server(db_path: str)` should call `from .transports.stdio import run_stdio_server_sync; run_stdio_server_sync(db_path)`
       - Ensure no print() statements — all output via logging to stderr
    6. CRITICAL: Verify NO `print()` calls anywhere in stdio.py. All logging must use `logger = get_logger(__name__)` which writes to stderr.
